CHAPTER A Closer Look at 5 Instruction Set Architectures
The instruction consists of opcodes and operands.
The opcodes specify the operations to be executed.
The operands specify register or memory locations of 
data.
The instruction sets are differentiated by 
the following features:
1- Operand storage in the CPU (data can be stored in  
stack structure or in registers)
2 – Number of operands per instruction.
3 – Operand location ( register to register, register to 
memory ……).
4 – Operations.
5 - Type and size (operands can be addresses, numbers , 
or characters).
